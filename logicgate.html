Introduction


    Boolean functions may be practically implemented by using electronic gates. The following points are important to understand.

  1. Electronic gates require a power supply.

  2. Gate INPUTS are driven by voltages having two nominal values, e.g. 0V and 5V representing logic 0 and logic 1 respectively.

  3. The OUTPUT of a gate provides two nominal values of voltage only, e.g. 0V and 5V representing logic 0 and logic 1 respectively. In general, there is only one output to a logic gate except in some special cases

  4. There is always a time delay between an input being applied and the output responding.

    Logic Gate

  1. The logic gates are the main structural part of a digital system.

  2. Logic Gates are a block of hardware that produces signals of binary 1 or 0 when input logic requirements are satisfied.

  3. Each gate has a distinct graphic symbol, and its operation can be described by means of algebraic expressions

  4. The seven basic logic gates includes: AND, OR, XOR, NOT, NAND, NOR, and XNOR.

  5. The relationship between the input-output binary variables for each gate can be represented in tabular form by a truth table.

  6. Each gate has one or two binary input variables designated by A and B and one binary output variable designated by x.

AND gate

The AND gate is so named because, if 0 is called "false" and 1 is called "true," the gate acts in the same way as the logical "and" operator. The output is "true" when both inputs are "true." Otherwise, the output is "false." In other words, the output is 1 only when both inputs one AND two are 1.

OR gate

The OR gate gets its name from the fact that it behaves after the fashion of the logical inclusive "or." The output is "true" if either or both of the inputs are "true." If both inputs are "false," then the output is "false." In other words, for the output to be 1, at least input one OR two must be 1. The NAND gate operates as an AND gate followed by a NOT gate. It acts in the manner of the logical operation "and" followed by negation. The output is "false" if both inputs are "true." Otherwise, the output is "true."

The NOR gate is a combination OR gate followed by an inverter. Its output is "true" if both inputs are "false." Otherwise, the output is "false."
